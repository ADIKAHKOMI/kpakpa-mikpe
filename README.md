#  Projet HTML – Site Web Personnel

##  Objectif
Ce projet consiste à créer un site web personnel en **HTML et CSS** afin de présenter :
- Mon parcours
- Mes compétences
- Mes projets
- Mes coordonnées

##  Structure du site
- `index.html` : Page d’accueil
- `apropos.html` : Présentation / Parcours
- `competences.html` : Compétences
- `projets.html` : Projets réalisés
- `contact.html` : Coordonnées

##  Mise en page
- En-tête avec menu de navigation
- Pied de page avec informations de contact
- Intégration d’images et icônes
- Styles CSS pour harmoniser couleurs, polices et marges

##  Validation
Le site est validé avec l’outil [W3C Validator](https://validator.w3.org/) afin de garantir la conformité aux standards HTML et CSS.
